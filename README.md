Конвертер температуры C -> F.

Программа для быстрого перевода градусов Цельсия в Фаренгейты с подсказками по погоде.

Как пользоваться:
1. Запусти `Конвертер температуры покруче.py`.
2. Введи число.
3. Радуйся результату!
4. Если надоело — напиши "стоп".
